**02**





**准备工作**









1. **微信小号**

   - 使用微信小号登录机器人，避免触发主账号风控。
   - 注意：严格遵守微信的用户协议，不用于非法途径。

2. **服务器环境**

   - 推荐使用 Ubuntu 20.04 或 22.04。
   - 配置需求极低，1核1G内存的服务器即可。
   - 可以选择国内外主流云服务商，如腾讯云、阿里云，或者便宜的香港服务器。（我是咸鱼22块/年钱淘香港服务器）

3. **获取 API**

   - 注册 智谱AI 或类似平台。（注册就有免费额度）

   - 获取 API Key，用于机器人智能问答功能。

**03**





**搭建机器人**









安装必要工具在服务器上安装 Python 和必要工具：

**sudo apt update && sudo apt install python3 python3-pip git ffmpeg -y**



克隆项目代码从 GitHub 或其他来源获取微信机器人项目代码：

**git clone https://github.com/your-repo/chatgpt-on-wechat.git**

**cd chatgpt-on-wechat**



安装依赖使用 pip 安装项目所需的依赖：

**pip3 install -r requirements.txt**



配置文件编辑项目目录中的 config.json 文件，替换为你的 API Key 和其他信息：

**{**

 **"zhipu_ai_api_key": "你的API Key",**

 **"zhipu_ai_api_base": "https://open.bigmodel.cn/api/paas/v4",**

 **"character_desc": "我是一个智能微信助手。",**

 **"temperature": 0.7**

**}**

**运行机器人**







###  

启动程序在项目目录中运行以下命令：

**python3 app.py**

程序会生成一个**二维码**。
