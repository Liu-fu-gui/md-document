## 网页加速器
【网页加速器】是一个及其强大的油猴脚本，利用浏览器的预加载技术，加速打开网页，打开网页速度加快 60% 以上。

原理：当您将鼠标悬停在链接上时，有 50％ 的机会单击链接，此时加速器会自动开始预加载页面到内存中，当您实际单击时（平均在 400 毫秒后），页面直接从内存中读取，节省了从网络下载的时间。
```
chrome-extension://dhdgffkkebhmkfjojejmpbldmpobfkfo/ask.html?aid=ef83c9c1-984e-4c59-a93e-956c5d683a71
```

## Mactype助手
Windows下的浏览器浏览网页时文字往往发虚，颜色很淡，看不清楚。有了它可以让浏览器中显示的文字更加清晰，支持Chrome ，360 ，QQ ，Firfox ，Edge 等浏览器。
```
https://greasyfork.org/zh-CN/scripts/436451-mactype%E5%8A%A9%E6%89%8B
```
强烈建议配合 XHei 字体使用，效果极佳。 字体下载地址(opens new window)
https://www.lanzouw.com/font
字体安装完成后在 Chrome 设置中搜索 “字体”，找到 自定义字体，将选项里的四个字体均设置为 XHei Intel
